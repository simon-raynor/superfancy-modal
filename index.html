superfancy-modal

Some examples of what can be achieved with modern CSS, every animation on this page is created with pure CSS: transitions, gradients and masks.

A small amount of JS is used to open/close the modals and to pass in the pointer position so that it can be used in the animations. A11y needs work, using <dialog> will eventually make all that easy but support is still spotty at best so I've used <div>s. There should at least be a prefers-reduced-motion: reduce so that they're not totally disrespectful of others' needs.

The code is up at https://github.com/simon-raynor/superfancy-modal, you are welcome to use it, adapt it, learn from it. The basic approach of using gradients and masks for more interesting popups should apply anywhere you can currently do a simple fade. As with all motion design type stuff it behooves you to not overdo things; I'm not sure most of these example could be used in a banking or healthcare app, nor would you want these effects on common actions you might be doing several times in a row.
