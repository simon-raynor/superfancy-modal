superfancy-modal

Some examples of what can be achieved with modern CSS; every animation on this page is created with CSS transitions, transforms, gradients and masks.

A small amount of JS is used to open/close the modals and to pass in the pointer position so that it can be used in the animations. A11y needs work, using <dialog> will eventually make that easy but support is still spotty at best so I've used good ol' <div>s for now. There should at least be a prefers-reduced-motion: reduce so that these are not totally disrespectful of others' needs. Performance seems pretty good, best for me in Chrome on my phone but ok everywhere I've tried. If it works on iOS it'll likely work well, will update once I find someone to test it :D

The code is up at https://github.com/simon-raynor/superfancy-modal, you are welcome to use it, adapt it, learn from it. The basic approach of using gradients and masks for more interesting popups should apply anywhere one can currently do a simple fade. It also doesn't have to be modals: animated pie charts using `conic-gradient` seems trivial at this point. Perhaps a loading screen that progressively uncovers or switches between some artwork? I've seen some cool effects in the wild recently and am excited to explore them myself.
